# Commercial Automation System

A robust commercial automation system specifically designed for use in pharmacies. This application ensures seamless control over products, suppliers, and users.

## Features

- **Product Management**: Efficiently manage and monitor your inventory.
- **Supplier Management**: Keep track of your suppliers and their products.
- **User Management**: Administer user roles and access.

## Technologies Used

- **Version Control**: Git for tracking changes and collaboration.
- **UI Manipulation**: Facelets for creating dynamic and reusable components.
- **Front-End Development**:
  - XHTML for structuring the content.
  - CSS for styling and layout.
  - JavaScript, jQuery, and PrimeFaces for interactive and rich web interfaces.
- **Back-End Development**:
  - JSF (JavaServer Faces) for building user interfaces.
  - AJAX for asynchronous web page updates.
  - MySQL for database management.
  - JavaScript for server-side scripting.

## Agile Methodology

- **Scrum**: Implemented Scrum for iterative development.
- **Trello**: Used Trello for task management and workflow organization.

## Testing

- **JUnit**: Comprehensive unit testing to ensure code reliability and performance.
